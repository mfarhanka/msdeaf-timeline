# images/

This directory is intended to hold historical photos for the MSDeaf Timeline.

## Naming Convention

Each image file should match the `image` field in `data.json`, for example:

- `1963-formation.jpg`
- `1965-championship.jpg`
- `1969-wfd.jpg`
- `1977-deaflympics.jpg`
- `1985-badminton.jpg`
- `1991-asean-games.jpg`
- `1997-futsal.jpg`
- `2001-complex.jpg`
- `2005-medal.jpg`
- `2010-youth.jpg`
- `2013-jubilee.jpg`
- `2017-apd-games.jpg`
- `2021-virtual.jpg`
- `2024-champions.jpg`

## Fallback

If an image file is missing, the timeline will automatically display a styled year-based
placeholder instead of a broken image.
